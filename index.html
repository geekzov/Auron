About Auron

Welcome to the official page of Auron programming language, designed for efficient web and mobile application development.

Key Features

Multi-paradigm Support
Strong, Static Typing
Type Inference
Built-in Concurrency
Native JSON Support
Cross-platform Compatibility

Sample Code

fn main() {
    let greeting = "Hello, Auron!";
    print(greeting);

    for i in range(1, 5) {
        print("Count: " + str(i));
    }

    async fn fetch_data(url: string) -> string {
        await sleep(1000);
        return "Data from " + url;
    }

    let urls = ["https://api1.com", "https://api2.com"];
    let results = await Promise.all(urls.map(fetch_data));

    for result in results {
        print(result);
    }
}
