About Auron

Welcome to the official page of the Auron programming language. Auron is a modern, multi-paradigm language designed for efficient web and mobile application development.

Key Features

Multi-paradigm Support
Strong, Static Typing
Type Inference
Built-in Concurrency
Native JSON Support
Cross-platform Compatibility

Sample Code

fn main() {
    let greeting = "Hello, Auron!";
    print(greeting);

    for i in range(1, 5) {
        print("Count: " + str(i));
    }

    // Demonstrating concurrency
    async fn fetch_data(url: string) -> string {
        // Simulating network request
        await sleep(1000);
        return "Data from " + url;
    }

    let urls = ["https://api1.com", "https://api2.com"];
    let results = await Promise.all(urls.map(fetch_data));
    
    for result in results {
        print(result);
    }
}
